IsObject

Returns a non-zero number if the specified value is an object.

IsObject := IsObject(ObjectValue)

Parameters

ObjectValue

A object stored in a variable, returned from a function, stored in another object or written directly.

Return Value

This function returns a boolean value representing whether the specified value is an object or not.

Related

Objects

Examples

object := {key: "value"}

if IsObject(object)
    MsgBox "This is an object."
else
    MsgBox "This is not an object."
