IsLabel

Returns a non-zero number if the specified label exists in the script.

IsLabel := IsLabel(LabelName)

Parameters

LabelName

Type: String

The name of a subroutine, hotkey, or hotstring (do not include the trailing colon(s) in LabelName).

Return Value

Type: Integer (boolean)

This function returns 1 (true) if the specified label exists, or 0 (false) if not.

Remarks

This function is useful to avoid runtime errors when specifying a dynamic label in functions such as Gosub, Hotkey, SetTimer and Menu.Add.

Related

Labels

Examples

#1

if IsLabel("Label")
    MsgBox "Subroutine exists"
else
    MsgBox "Subroutine doesn't exist"

Label:
return

#2

if IsLabel("^#h")
    MsgBox "Hotkey exists"
else
    MsgBox "Hotkey doesn't exist"

^#h::return

#3

if IsLabel("::btw")
    MsgBox "Hotstring exists"
else
    MsgBox "Hotstring doesn't exist"

::btw::by the way
