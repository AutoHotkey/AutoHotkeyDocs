FileRecycleEmpty

Empties the recycle bin.

FileRecycleEmpty DriveLetter

Parameters

DriveLetter

Type: String

If omitted, the recycle bin for all drives is emptied. Otherwise, specify a drive letter such as C:\

Error Handling

ErrorLevel is set to 1 if there was a problem or 0 otherwise.

Remarks

This functions requires that MS Internet Explorer 4 or later be installed.

Related

FileRecycle, FileDelete, FileCopy, FileMove

Examples

#1

FileRecycleEmpty "C:\"
