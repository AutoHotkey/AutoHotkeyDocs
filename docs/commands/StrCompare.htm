StrCompare

Compares two strings alphabetically.

Result := StrCompare(String1, String2 ,  CaseSensitive := false)

Parameters

String1, String2

Type: String

The strings to be compared.

CaseSensitive

Type: Integer (boolean)

If the parameter CaseSensitive is omitted or 0 (false), the comparison is not case sensitive (the method of insensitivity depends on StringCaseSense); otherwise, the case must match exactly for the strings to be consider identical.

Return Value

Type: Integer

To indicate the relationship between String1 and String2, this function returns,

  • 0, if String1 is identical to String2.
  • a positive integer, if String1 is greater than String2.
  • a negative integer, if String1 is less than String2.

Remarks

This function is commonly used for sort callbacks.

Related

StringCaseSense, Sort.

Example

MsgBox StrCompare("Abc", "abc") ; Returns 0
MsgBox StrCompare("Abc", "abc", true) ; Returns -1
