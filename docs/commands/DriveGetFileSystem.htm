DriveGetFileSystem

Returns the type of the specified drive's file system.

FileSystem := DriveGetFileSystem(Drive)

Parameters

Drive
The drive letter followed by a colon and an optional backslash, or a UNC name such \\server1\share1.

Return Value

This function returns a string representing the type of Drive's file system: FAT, FAT32, NTFS, CDFS (typically indicates a CD), or UDF (typically indicates a DVD). If Drive does not contain formatted media, the return value is blank and ErrorLevel is set to 1.

ErrorLevel

ErrorLevel is set to 1 if there was a problem or 0 otherwise.
