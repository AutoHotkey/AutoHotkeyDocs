AutoHotkey v2

AutoHotkey v2 aims to improve the usability and convenience of the language and command set by sacrificing backward compatibility. As the feature set, behaviour and syntax aren't yet set in stone, it is not recommended for general use.

Downloads

These downloads include AutoHotkey.exe and some miscellaneous function scripts.

  • Unicode 32-bit - for 32-bit or 64-bit systems.
  • Unicode 64-bit - for 64-bit systems only.
  • Source Code - download via git or click the "ZIP" button at github.

Microsoft Visual C++ 2010 has been used to build these executables. Windows XP and later are supported.

Source code is available at github.
